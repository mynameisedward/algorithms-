Бинарный поиск - эффективный способ найти элемент в отсортированном массиве

Важнейшее условие: отсортированность массива

Бинарный поиск можно применить к любой задаче, которую можно свести к поиску монотонной функции
    Монотонная функция - это функция, значение которой либо всегда возрастает либо всегда убывает
        т.е значение функции будет всегда больше предыдущего значения
                                     либо меньше предыдущего значения
