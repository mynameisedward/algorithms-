Priority Queue - это очередь, с каким-то приоритетом
    У кого приоритет больше, тот идет первым

Рассмотрим реализацию через Binary Heap


Binary Heap бывает двух видов:
    Min heap
    Max heap



Чтобы дерево можно было назвать Binary Heap, должно быть выполнено 2 условия,
    1) дерево должно быть complete
    2)  a: Max-heap  Значение в каждой ноде должно быть больше или равно чем значения в детях
        b: Min-heap  Значение в каждой ноде должно быть меньше или равно чем значения в детях

Что такое complete?
    Это когда все этажи дерева заполнены
        Последний этаж может быть не заполнен



Выбрать наибольший элемент в Max heap - O(1)
Вставка элемента - O(log n)
Удалить наибольший элемент - O(log n)
Сделать из массива Heap - O(n)  Hippify



Binary Heap - это будет массив
    i >= (i * 2) + 1
    i >= (i * 2) + 2

    Родитель: Math.floor((i - 1) / 2)
