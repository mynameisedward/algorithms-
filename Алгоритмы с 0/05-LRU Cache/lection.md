Задача, в которой нужно написать Кэш LRU(least recently used)
    Кэш который соответствует политике LRU

Допустим в кэше размером 3 хранится три ключа: 1, 2, 3
    Нужно добавить в кэш ключ "4"

    Какой из трех ключей нужно убрать?
        LRU - нужно убрать тот, который ДАВНО использовался 

    ------------------------------------------->
    1 1 1 1 1 1      2      3                 удаляем из кэша "1" т.к давно использовали добавляем 4




Используется много где:
    - Браузеры - хранит наиболее посещаемые страницы
    - Операционные системы
    - Базы данных
    - Мобильные приложения


Реально полезная структура данных


Операции в LRU Cache работают в среднем за О(1)
