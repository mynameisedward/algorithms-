Одна из популярных вариаций бинарного дерева - бинарное дерево поиска

Что такое бинарное дерево поиска?
    Значит, что для каждой ноды, значение слева будет меньше, а значение справа будет больше

Когда мы обходим бинарное дерево поиска - мы обходим только половину дерева, половину нод

Поиск происходит за O(log(n)) в лучшем случае
И за O(n) в худшем случае
